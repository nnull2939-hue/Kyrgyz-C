Kyrgyz-C (v0.4 — Толук стандарт) 🇰🇬
Эмне бул? (Что это?)

Я Давид, учусь в 7 классе. Мне 14 лет. Я долго думал, с чего начать свой путь в программировании, и решил сделать что-то полезное для ребят из Кыргызстана.

Kyrgyz-C — это перевод стандартного языка Си (C11) на кыргызский язык. Я потратил немало времени, чтобы обдумать логику и перевести ключевые слова так, чтобы они звучали понятно.
Эмне үчүн? (Зачем это нужно?)

Многие дети у нас в стране не знают английский на уровне профи, и из-за этого им сложно учить программирование. Я хочу, чтобы любой школьник мог сразу учить логику на родном языке.

    Это настоящий Си, на котором работают серверы и даже «китайские чайники» (микроконтроллеры).

    Никакого ООП и тяжелых штук — только чистый C.

    Разработано и протестировано на BlackArch Linux.

Кантип иштейт? (Как это работает?)

Всё просто: я использовал препроцессор, чтобы заменить английские команды на наши.
Вместо if пишем эгерде, вместо int — бүтүн, вместо while — кайтала.



Кантип иштетсе болот? (Как запустить?)

    Скачай kyrgyz.h.

    Подключи его в своем коде: #include "kyrgyz.h".

    Компилируй через обычный gcc:
    Bash

    gcc main.c -o kyrgyz_prog -lm
